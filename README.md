Amazon Sales Dashboard | Power BI Project
Project Overview

This project analyzes Amazon sales data (2011–2014) using Power BI to uncover key business insights related to sales performance, profit trends, delivery time, and regional analysis. The dashboard is fully interactive and designed for data-driven decision-making.

Dataset Information

* Source: Amazon Sales Dataset

* Records: 3203 → cleaned to 1611 unique orders

* Key Fields: Order Date, Ship Date, Category, Sales, Profit, Quantity, City, State

Data Cleaning (Power Query)

* Removed null values, errors, and blank rows

* Removed duplicate Order IDs

* Dropped unnecessary columns

* Split Geography into City & State

* Extracted Year from Order Date

* Created Delivery Days column
